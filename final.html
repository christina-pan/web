This program times your typing.

  • How it works

    We provided a quote from a book in a GUI. Here we used Tkinter and Timeit for GUI and typing speed, respectively. First, we imported all the libraries mentioned above and created a speed testing window using Tkinter. Next, with Timeit, we recorded the time spent on each provided passage. Then, with a series of strings, the program with calculate the WPM (Words Per Minute) when prompted.
